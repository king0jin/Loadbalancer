# nginx를 이용한 LoadBalancer수행을 위한 간단한 flask 웹 애플리케이션 생성

### flask 패키지 설치
+ **pip install flask**


### 간단한 flask 웹 애플리케이션 생성
app1.py ~ app5.py
+ 포트번호 다르게 부여 : 5001 ~ 5005


### 웹 애플리케이션 실행
웹 애플리케이션이 위치하고 있는 디렉토리에서 실행
+ **python app1.py**
+ ~
+ **python app5.py**
